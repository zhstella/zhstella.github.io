# Contributing

This is a public project. Anyone can propose edits through GitHub's standard fork-and-pull-request workflow.

## How to edit

1. Fork the repository.
2. Make changes in your fork.
3. Open a pull request back to `jamesbbaker/sound_of_popularity`.
4. A maintainer can review and merge the pull request.

## Local preview

```sh
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000/
```

## Notes

- The app is static HTML, CSS, and JavaScript.
- There is no build step.
- GitHub Pages serves the app from the `main` branch root.
